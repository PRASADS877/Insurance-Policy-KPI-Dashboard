# 🛡️ Insurance Policy KPI Dashboard 2026

## 📊 Project Overview
An interactive KPI Dashboard built in Microsoft Excel analyzing 
5,000 insurance policy records across 5 tables.

## 🎯 Key KPIs Tracked
- Total Policies — 5,000
- Total Customers — 5,000
- Policy Expire This Year — 453
- Total Claim Amount — ₹251.38 Cr
- Premium Growth Rate — +12.25%
- Claim Status Distribution (Approved/Pending/Denied)
- Payment Status Distribution (Successful/Failed)
- Age Bucket Wise Policy Count
- Gender Wise Policy Count
- Policy Type Wise Count

## 🛠️ Tools and Skills Used
- Microsoft Excel
- Power Query — Data cleaning and merging
- Pivot Tables — KPI calculations
- Pivot Charts — Data visualization
- Slicers — Interactive filtering
- DAX Formulas — Custom calculations

## 📁 Dataset
- 5 tables — 5,000 rows each
- Customer Information
- Policy Details
- Claims
- Payment History
- Additional Fields

## 🔄 Process
1. Loaded all 5 tables into Power Query
2. Cleaned data — removed duplicates, handled nulls
3. Added calculated columns — Age Bucket, Policy Duration, Expiry Flag
4. Summarized Claims and Payment History
5. Merged all 5 tables into Master Data
6. Built 10 KPIs using Pivot Tables and formulas
7. Created 6 interactive charts
8. Connected 3 slicers for dynamic filtering
9. Designed professional dark navy dashboard
